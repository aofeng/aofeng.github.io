单元测试是保证代码质量不可缺少的手段，熟悉Java的童鞋非常清楚：使用JUnit测试框架，再加上Eclipse集成的JUnit插件，在开发的过程中进行单元测试非常方便和顺手。
PHP也有PHPUnit，Eclipse也推出了PHPEclipse，却没有支持PHPUnit的插件，所以使用起来还得费一番功夫。下面和大家分享在PHPEclipse如何高效地进行单元测试。

一、安装PHPUnit | Install PHPUnit

二、在Eclipse中设置PHPUnit | PHPUnit setting

打开菜单：Run –> External Tools –> External Tools Conrigurations…，然后新建一个Program。如下图：

说明：
1）新建一个 Program，命名为 PHPUnit（名称可自定义）。
2）Location 中填写phpunit所在的绝对路径。
3）Working Directory 中填写 ${workspace_loc}。
4）Arguments 中填写${resource_log}。
