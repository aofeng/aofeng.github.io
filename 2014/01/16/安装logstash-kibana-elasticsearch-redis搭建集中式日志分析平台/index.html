本文是参考logstash官方文档实践的笔记，搭建环境和所需组件如下：

  • Redhat 5.7 64bit / CentOS 5.x

  • logstash 1.3.2 (内带kibana)
  • elasticsearch 0.90.10
  • redis 2.8.4
