umask导致的问题

我的开发服务器默认的umask设置为0037，结果安装后，在root用户下可以正常地使用mysql.connector模块，但是切换至其他的普通账号时，会报错：
ImportError: No module named mysql.connector

解决方法：
在执行python setup.py install之前，先执行umask 0022。
